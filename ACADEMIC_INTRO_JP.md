# 🧠 AI教示設計（AI-Kyōji-Sekkei）学術案内  
_人とAIが「共に教え合う」ための知能設計アーキテクチャ_

---

## 1. 概要（Overview）

**AI教示設計（AI-Kyōji-Sekkei）** は、  
「AIに教える方法をAIに教える」ことを目的とした、  
**人間とAIの協働的学習構造**のためのフレームワークです。

この体系は、より大きな概念群である **SOVOS（Structured Operating Visioning System）** に基づき、  
AIと人間が「共に設計し、共に改善する」ためのテンプレート構造を提供します。

本プロジェクトでは、従来の「AIを使う」から一歩進み、  
**AIと共に教示構造を再構築する**ことを目的としています。  
その中核となるのが「リバース・ティーチング（Reverse Teaching Methodology）」、  
すなわち、ユーザーがAIに命令を与えるのではなく、  
AIと共に命令を**設計していく**という発想です。

---

## 2. 研究上の位置づけ（Research Positioning）

AI教示設計は、以下の研究分野にまたがる横断的アーキテクチャです。

| 研究領域 | 関連テーマ |
|-----------|-------------|
| 人間中心AI | インタラクティブな教示設計理論 |
| 認知システム工学 | メタプロンプト設計／逆プロンプト設計 |
| 機械学習 | 自己整合性・リフレクション・再教示モデル |
| AI倫理・教育工学 | 共創的AI教育と責任あるAI活用 |

従来の **RLHF（人間フィードバックによる強化学習）** が  
固定的な報酬構造を用いるのに対し、  
AI教示設計は**対話的・継続的な教示ループ**を通じて  
動的にAIの指示構造を共に設計していきます。

これは産業用ロボットの「ティーチング操作」から着想を得ており、  
固定プログラムではなく、**動作座標を人間が教え、AIが再学習する**構造を再現しています。

---

## 3. 方法論の概要（Methodological Summary）

AI教示設計は、以下の三層構造によって構成されます。

| レイヤ | 名称 | 主な役割 |
|--------|------|----------|
| **コア層（SoE OS Kernel）** | 推論フローの安定基盤 | 指示文解析・ループ制御 |
| **教示層（Kyōji Engine）** | 教示設計・再教示・リフレクション | 共設計・再学習・構造化 |
| **応用層（Agent Templates）** | ユーザー向けインターフェース | 創作支援・教育応用 |

このフレームワークは、以下の4ステップのループで動作します。

1. **教示（Teach）** – ユーザーが目的を与える  
2. **反省（Reflect）** – AIが最適プロンプトを自己生成し、理由を説明する  
3. **再教示（Re-Teach）** – ユーザーが修正・再設計を加える  
4. **安定化（Stabilize）** – 改良された構造を次の世代に引き継ぐ  

このループによって、AIと人間は**対話的共進化（Co-Evolution）**を遂げます。  
学習データではなく、「共に考えるプロセス」そのものが知能成長の原動力となります。

## 4. 既存研究との関係（Relation to Existing Research）

AI教示設計は、既存の複数のAI研究領域を統合する実践的枠組みです。

| 既存理論 | 本体系との関係 |
|-----------|----------------|
| **RLHF** | 報酬学習を越え、対話を通じた連続的な教示へ拡張 |
| **Reflection / Self-Consistency** | リフレクションを明示的設計段階として導入 |
| **Interactive Instruction Tuning** | 人間を「設計相手」として含む共教示構造を標準化 |
| **Reverse Prompt Engineering** | 解析手法ではなく、共創的プロンプト設計法として再定義 |

これにより、理論的研究を**教育実装レベルに転換する**ことが可能になります。

本プロジェクトは、**教育・研究目的限定**で設計されています。  
ライセンスは **MIT＋CC BY 4.0** に基づき、  
世界中の教育機関・研究機関が自由に研究・学習に利用できることを目的とします。

倫理的な利用指針として：
- 自律AIの再訓練や外部モデルの改変に用いないこと

このテンプレートは**自己学習AIの開発ツールではなく、AIを教えるための教材**です。  
目的は「AIと共に考える力」を養うことにあります。
